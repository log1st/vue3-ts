# @urrobot/app

## Установить и запустить локально
1. Зависимости
```bash
yarn install
```
2. Переменные среды
```bash
cp .env.example.development .env.development.local
# При желании можно заменить значения внутри .env.local 
```
3. Запустить
```bash
yarn serve
```

## Установить и собрать продакшн-билд
1. Зависимости
```bash
yarn install
```
2. Прокинуть переменные среды в зависимости от метода сборки
3. Собрать
```bash 
yarn build
# артефактом будет директория ./dist, в неё должен смотреть веб-сервер
cp ./dist/* <ваша директория для веб-сервера>
```
